# Project ICE v0.5

This repository is an AI-first engineering specification for a browser-based hockey game.

Guiding principles:
- Requirements before implementation
- Verification before optimization
- Deterministic simulation
- Modular architecture
- Explicit hockey knowledge

Read PROJECT_ICE_PRD.md first, then review the architecture, gameplay, and handoff notes for the current design direction.
